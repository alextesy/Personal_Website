    Web 1.0

    Static

    There is only limited interaction between sites and web users. Simply an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback.

    Example

    Web 2.0

    Writable

    Unlike Web 1.0, Web 2.0 facilitates interaction between web users and sites, so it allows users to interact more freely with each other. Web 2.0 encourages participation, collaboration, and information sharing.

    Example

    Web 3.0

    Executable

    Web 3.0 is a semantic web which refers to the future. In Web 3.0, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users.

    Example

    Lets Examine the next used case: Searching a specific query.

    Searching A specific query in Web 1.0 would be extremely difficult. First of all you would have to know the name of the Website (No Search Engines) that specializes on the domain of the querry. When the user will enter the site he probably will be able to search there but there won't be any other options - no posting comments, no sharing, nothing Interactive. Only readible - static pages.

    In web 2.0 you can easily visit Google (or any other) search-engine and search your query there. It will provide with multiple sites that can answer your question. In most of them you will be able to comment, search for recommndations, share on social media and many more options.

    Web 3.0 will probably know your query before you'll type it. It will prepare the best fitting answers to you (based on your preferences and studying your behaviour) and deliver it to you automatically.
